# 🛡️ SoftNexis Cyber Security Internship Projects

This repository contains all internship tasks and assignments completed during the **SoftNexis Cyber Security Internship Program**.

The internship focuses on cybersecurity fundamentals, virtual lab setup, networking concepts, reconnaissance techniques, vulnerability assessment, and professional security tools used in ethical hacking and penetration testing.

- IP Address Identification

# ✅ Task 2 — Network Scanning & Enumeration using Nmap

Performed network reconnaissance using **Nmap** to discover live hosts, open ports, running services, and operating system information.

### Features

- Host Discovery
- Port Scanning
- Service Enumeration
- Version Detection
- Operating System Detection
- Full Port Scan
- Network Verification

### Nmap Commands Used

```bash
nmap <target-ip>

nmap -A <target-ip>

nmap -sV <target-ip>

nmap -p- <target-ip>
```

### Concepts Learned

- Reconnaissance
- Enumeration
- Open Ports
- Network Services
- TCP/IP Fundamentals
- Attack Surface Identification

### Tools Used

- Kali Linux
- Nmap
- Metasploitable 2
- VirtualBox

# ✅ Task 3 — Web Application Vulnerability Assessment using OWASP ZAP

Performed automated vulnerability assessment on the Metasploitable web application using **OWASP ZAP**.

### Features

- Target Website Crawling
- Passive Scan
- Active Scan
- Vulnerability Detection
- Risk Classification
- Alert Analysis
- Security Report Review

### Vulnerabilities Identified

- Cross-Site Scripting (XSS)
- Missing Anti-CSRF Tokens
- Security Header Issues
- Information Disclosure
- Content Security Policy Issues
- Server Information Leakage

### Tools Used

- OWASP ZAP
- Kali Linux
- Metasploitable 2
- Firefox Browser

### Concepts Learned

- Web Application Security
- Vulnerability Assessment
- Automated Security Testing
- Risk Analysis
- Security Misconfigurations
- OWASP Top 10 Awareness
